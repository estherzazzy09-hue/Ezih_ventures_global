## 🔧 How to Customize

### Add Your Product Images

1. **Create assets folder structure:**
   ```
   assets/
   ├── products/
   │   ├── product1.jpg
   │   ├── product2.jpg
   │   ├── product3.jpg
   │   └── ...
   ```

2. **Update index.html** - Replace placeholder images:
   - Find: `src="https://via.placeholder.com/300x250?text=Product+1"`
   - Replace with: `src="assets/products/product1.jpg"`

3. **Update script.js** - Update product modal images:
   - Find the `products` object in `script.js`
   - Update image URLs to your actual product images

### Customize Products

Edit the products array in `script.js`:
```javascript
const products = {
    1: {
        title: 'Your Product Name',
        image: 'assets/products/product1.jpg',
        category: 'Product Category',
        description: 'Your product description here'
    },
    // ... more products
}
```

## 📸 Adding Product Images

### Best Practices:

1. **Image Quality**: Use high-resolution product photos (1000x800px or larger)
2. **Consistency**: Keep product images in the same style/background
3. **File Format**: Use JPG or PNG format
4. **File Size**: Optimize images to load faster (<500KB per image)
5. **Naming**: Use clear names like `product-1.jpg`, `product-2.jpg`
